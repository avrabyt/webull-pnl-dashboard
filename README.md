# Webull P&L Dashboard

A mobile-optimized financial dashboard UI matching the Webull P&L interface design.

## Features

- **P&L Overview**: Display profit & loss metrics with date range selector
- **Trend Analysis**: Interactive line chart showing P&L trends over time
- **Asset Class Chart**: Donut chart showing P&L breakdown by asset class
- **P&L Calendar**: Calendar view with daily P&L values
- **Responsive Design**: Fully optimized for mobile and desktop devices
- **Custom Date Range Picker**: Modal for selecting custom date ranges

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Recharts (for charts)
- CSS3 (mobile-responsive)

## Color Scheme

- **Positive Values**: Green (#00C853)
- **Negative Values**: Red (#FF5252)
- **Primary Blue**: #2196F3
- **Background**: Light gray (#f5f5f5)
- **Cards**: White (#ffffff)
